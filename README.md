# 🛒 Mini E-Commerce Backend (Clean Architecture)

Backend de nivel empresarial diseñado para demostrar la implementación de patrones de arquitectura de software modernos y mejores prácticas en .NET 9.

## 🎯 Objetivos del Proyecto

El propósito principal es simular un entorno de producción real para dominar conceptos avanzados:

* **Arquitectura:** Clean Architecture (Domain-Centric) desacoplada.
* **Persistencia:** Entity Framework Core con enfoque **Code First**, manejo avanzado de relaciones y **UnitOfWork**.
* **Performance:** Implementación de **Lazy Loading** estratégico y **Caché Distribuido (Redis)** con patrón Decorator/Proxy.
* **Calidad:** Testing unitario con **xUnit & Moq**.
* **DevOps:** Containerización completa con **Docker & Docker Compose**.

## 🛠️ Stack Tecnológico

* **.NET 8** (C#)
* **SQL Server** (Persistencia)
* **Redis** (Caché)
* **Docker** (Infraestructura)
* **Entity Framework Core** (ORM)
